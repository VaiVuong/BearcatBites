# Bearcat Bites
Food and Drink recommendation for around the UC area built on ASP.NET

## Features
* Separate pages for Food and Drinks
* Sort by "Culture" to experience the diverse tri-county area

## Tech
* ASP.NET Core (MVC)
* language: C#, Java

## Prerequisites
* .NET 8
